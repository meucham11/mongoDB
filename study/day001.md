### DB 생성 : use aaa
#### 첫번째 방법
```MONGODB
> use SALES
> db.createCollections("employees",{capped:true, size:100000})     capped : 저장 공간의 재사용이 가능한 타입   size : collections의 최초 익스텐트 크기
> show collections
> db.employees.stats()
```
#### 두번째 방법




### Collections 이름 변경
```mongodb
> db.employees.renameCollections("emp")
```

### Collections 삭제
```mongodb
> db.emp.drop()
```
